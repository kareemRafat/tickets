# Project-Scoped Rules: Arabic RTL UI
1. All user-facing views, templates, dashboards, login pages, verification inputs, forms, validation errors, and alert messages must be in Arabic language only.
2. The HTML direction must be set to RTL: `<html lang="ar" dir="rtl">`.
3. Use the 'Cairo' Google Font for all Arabic text rendering to maintain a premium look.
4. Align sidebars to the right (`right-0`, `translate-x-full`) and content padding margins to the right (`sm:mr-64`).
5. Ensure responsive menus, buttons, and alerts are correctly configured for RTL.
